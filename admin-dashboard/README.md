# Presensia - Admin Dashboard (Next.js)

This folder contains the admin-dashboard skeleton using Next.js App Router and Tailwind CSS.

To run locally:

```bash
cd admin-dashboard
npm install
npm run dev
```

This is a UI skeleton. Connect to your backend by adding API calls and auth handling. Tailwind is configured via `tailwind.config.js` and global styles in `app/globals.css`.
